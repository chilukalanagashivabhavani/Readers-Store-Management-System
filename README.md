# Readers Store Management System

## Description
Readers Store Management System is a college mini project developed for Anurag University. This initial version is created using only HTML as part of the HTML module. It includes Home, Login, and Contact Us pages. CSS, JavaScript, and additional features will be added in future versions.

## Technologies Used
- HTML5

## Project Structure
- Home Page
- Login Page
- Contact Us Page
